# Research Toolkit

Tools and services Sibyl uses for research.

## Web Search

- **Tool:** Claude Code web search (built-in)
- **Scope:** Current events, market signals, announcements
- **Citation:** All results must be cited with URL
- **Rate limit:** No documented limit; respect ethical use

## Industry Reports

- **Gartner, Forrester, IDC:** Access via koad:io (if available)
- **White papers:** Usually available from vendor websites
- **Academic:** Google Scholar, SSRN

## Standards Bodies

- **W3C DIDs:** https://www.w3.org/TR/did-core/
- **W3C Verifiable Credentials:** https://www.w3.org/TR/vc-data-model/
- **NIST Digital Identity Guidelines:** https://pages.nist.gov/800-63-3/
- **IETF RFCs:** https://www.ietf.org/

## Competitive Intelligence

- **GitHub:** Public repos, stars, commits, issues
- **LinkedIn:** Company profiles, job postings, hiring trends
- **Crunchbase:** Funding, acquisitions, personnel
- **News aggregators:** TechCrunch, VentureBeat, The Block, CoinDesk

## Sourcing Discipline

- Always cite specific URL or document
- Note publication date
- Flag analysis vs. reporting (if commentary)
- Verify claims where possible

## Sources to Always Verify

- Vendor claims (competitors often overstate features)
- Funding announcements (may be inflated)
- Market size estimates (vary wildly by analyst)
- Adoption claims (look for evidence beyond press release)
