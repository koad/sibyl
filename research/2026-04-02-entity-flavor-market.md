## 1. What Roles Are Developers Actually Trying to Fill?

### The $60-90/mo Stack (Real Spending Data)

Indie hackers and solo developers are paying for:

- **Chat/Research:** ChatGPT Plus ($20), Gemini Advanced ($20)
- **API Credits:** OpenRouter (~$50/mo)

**Total typical spend: $60-150/mo per developer** — confirmed by multiple Indie Hackers posts (Mar 2026).

| 3 | **Executive Assistant** | Calendar, tasks, research synthesis | $20-50/mo |
| 4 | **Personal Memory** | Remembering project context across sessions | $10-30/mo |

**Source:** Indie Hackers spending threads, Orbator AI stack guide, Built This Week analysis

### The Memory Problem (Critical Gap)

Every search result on "AI agent memory" from 2026 identifies the same core issue:

> "Your AI agent forgets everything the moment a session ends. It processed 200 purchase orders last week and made the same mistake on number 201."
> — NewClaw Times, Apr 2026

**Market validation:**
- Mem0 raised funding for persistent memory layer

### Data Sovereignty Gap

From AI Journal (Apr 2026):
> "64% of enterprises already experimenting with agentic AI... the question is how to deploy autonomous systems that reason over sensitive data"

**What sovereign identity fixes:**
- Cryptographic agent identity (not API keys)
- Local-first = no data leaves infrastructure
- Agent-to-agent authentication without cloud dependency
- Zero cloud cost for compute (run on own hardware)

**Source:** AI Journal, Medium sovereign AI analysis, Vitalik's self-sovereign LLM setup (Apr 2026)
